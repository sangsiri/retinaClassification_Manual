วิธีการใช้งานไลน์บอท

ขั้นตอนที่ 1
เลือกเมนูในการใช้งานว่าจะเลือกการตรวจโรคเป็นโหมดใด โดยจะมีทั้งหมด 2 โหมด คือ
1.Fundus Mode
2.Swept-Source Mode
ขั้นตอนที่ 2
หลังจากเลือกโหมดแล้ว โดยในที่นี้เราจะขอยกตัวอย่างเป็นโหมด Fundus Mode แชทบอทจะตอบกลับมาว่า Fundus Mode OK
ขั้นตอนที่ 3
เมื่อแชทบอทตอบกลับมาว่า Fundus Mode OK แสดงว่าพร้อมสำหรับการใช้งานแล้ว ให้กดเลือกรูปจอประสาทตาที่จะส่งแล้วกดส่งได้เลย
ขั้นตอนที่ 4
เมื่อส่งรูปภาพไปแล้ว ตัวปัญญาประดิษฐ์จะใช้เวลาซักครู่ในการวินัจฉัยโรค แล้วจึงตอบกลับมาว่าเจอโรคหรือไม่พร้อมกับส่งรูปที่ทำการตรวจแล้วออกมา
โดยในภาพตัวอย่างนั้นเป็นภาพจอประสาทตาปกติ ปัญญาประดิษฐ์จึงตอบกลับมาว่า "NORMAL found!" คืออาการปกติ ไม่พบโรคนั่นเอง แล้วจึงส่งหลักฐานการตรวจกลับมาด้วย

ฟังก์ชั่นอื่นๆ

โหมดที่กำลังใช้งานอยู่
เมื่อกดที่คำสั่ง "โหมดที่กำลังใช้งานอยู่" ตัวแชทบอทก็จะตอบกลับมาว่าตอนนี้เป็นโหมดตรวจแบบไหนอยู่ เช่นในตัวอย่างคือ Fundus Mode
ผู้จัดทำ
เมื่อกดที่คำสั่ง "ผู้จัดทำ" ตัวแชทบอทจะพาผู้ใช้งานไปยังเว็บไซต์แสดงชื่อและภาพของผู้จัดทำและอาจารย์ที่ปรึกษา
